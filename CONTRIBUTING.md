# Contributing

This is a personal portfolio site. It's source-available for reference, but content changes
(bio, résumé, work history) are personal and won't take outside PRs. Bug reports for the site
itself (broken layout, accessibility issues, broken links) are welcome via
[Issues](../../issues).

## Local preview

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`. No build step — plain HTML/CSS/JS.

## Before opening a PR

- Check the page still renders correctly at mobile, tablet, and desktop widths.
- Run a link checker if you touched navigation or external links.

This project follows the [Code of Conduct](CODE_OF_CONDUCT.md).
